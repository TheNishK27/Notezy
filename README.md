# 🎓 Student Journey Guide

A web platform designed to help students access important academic resources like previous year question papers and college-related information.

🌐 Live Demo: https://student-journey-guide.vercel.app

---

## 🚀 Features

* 📄 Access previous year question papers
* 🎓 College-related information in one place
* 📥 Download resources easily
* 🖥️ Simple and clean user interface
* 🔐 Backend integration for handling data

---

## 🛠️ Tech Stack

* **Frontend:** EJS, HTML, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Other Tools:** Git, GitHub, Vercel

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/Deepak-kumar-2023/Student-Journey-Guide.git
```

2. Navigate to the project directory

```bash
cd Student-Journey-Guide
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file using `.env.sample`

```env
PORT=3000
MONGO_URI=your_mongodb_connection_string
```

5. Run the application

```bash
npm start
```

6. Open in browser

```
http://localhost:3000
```
